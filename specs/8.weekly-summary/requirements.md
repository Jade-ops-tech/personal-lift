# weekly-summary — 需求规格

## 概述

生成并展示每周总结：本周记录趋势、高频分类/标签、主要内容、新增想法、完成事项与下周建议跟进。

## 项目信息

- 项目名: personal-lift
- 架构类型: Turborepo monorepo

## 需求版本

| 日期 | 版本 | 说明 |
| ---- | ---- | ---- |
| 2026-06-17 | v1 | 初始需求 |

## 用户故事

- 作为用户，我想查看本周总结，以便回顾一周并规划下周要跟进的事。

## 功能需求

1. [F-001] 生成本周总结，内容含：本周记录趋势、高频分类、高频标签、主要工作/学习/生活内容、本周新增想法、本周完成事项、下周建议跟进事项。
2. [F-002] H5 总结页展示本周总结（与今日总结同页切换）。
3. [F-003] 总结持久化到 summaries（type=weekly, period_key=ISO 周，如 2026-W25），重复生成覆盖更新。

## 非功能需求

- 性能: 概括允许稍高延迟，提供生成中状态。
- 安全: 生成/查询经 protectedProcedure（security.md）。

## 验收标准

- [ ] [AC-001] 用户可查看本周总结（PRD H5 验收 8）。
- [ ] [AC-002] 周总结覆盖高频分类/标签与完成事项、下周建议。
- [ ] [AC-003] 本周无记录时给出合理空态。
